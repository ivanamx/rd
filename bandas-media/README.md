# Archivos multimedia de bandas

Coloca aquí las fotos y muestras de audio de cada banda siguiendo esta estructura:

```
bandas-media/
  {id-de-banda}/
    cover.jpg          ← foto de perfil
    muestra-1.mp3      ← canción de ejemplo
    muestra-2.mp3
```

Los IDs coinciden con los definidos en `bandas.json` (ej. `los-del-desierto`, `neon-saltillo`).
